Parcel Tracking App
A lightweight Java-based application designed to manage and track the status of delivery parcels. This project demonstrates core Object-Oriented Programming (OOP) principles through a simple, command-line or GUI-driven interface for logistics management.

Features
Add New Parcels: Register unique tracking IDs and recipient details.

Status Updates: Update parcel locations and delivery stages (e.g., Shipped, In Transit, Delivered).

Search Functionality: Quickly find a parcel's status using its tracking number.

Data Persistence: (Optional/If applicable) Saves tracking history for future sessions.

Technologies Used
Language: Java

Tools: JDK 8 or higher

IDE: IntelliJ IDEA / Eclipse / VS Code

Installation & Setup
To run this project locally, follow these steps:

Clone the repository:

Bash
git clone https://github.com/Haniatahir05/Parcel-Tracking-App.git
Navigate to the directory:

Bash
cd Parcel-Tracking-App
Compile the Java file:

Bash
javac ParcelTrackingApp.java
Run the application:

Bash
java ParcelTrackingApp
Usage
Once the program is running, follow the on-screen prompts to:

Input the parcel details.

Assign a tracking number.

Check the current status of any package in the system.
